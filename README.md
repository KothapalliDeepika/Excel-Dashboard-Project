📊 Employee Analytics Dashboard using Microsoft Excel

Welcome to my Employee Analytics Dashboard project!

This dashboard was created as part of my Data Analytics learning journey at WhiteScholars Academy. The main objective of this project was to analyze employee data and present meaningful insights through an interactive and user-friendly dashboard.
Using Microsoft Excel, I transformed raw employee data into visual reports that help understand workforce trends and support better decision-making.

Project Highlights

This dashboard provides insights into:

- Employee Count by Department
- Department-wise Average Salary
- Employee Joining Year Trend
- Performance Category Analysis
- Bonus Eligibility Analysis
- City-wise Employee Distribution
The dashboard also includes interactive Slicers and KPI Cards, allowing users to filter and explore the data easily.
